Regression Lines in Regular Polygons: A Statistical Curiosity

March 15, 2025 By Diganta Bhattacharya

During my preparation for the B.Stat comprehensive viva, I encountered an intriguing statistical problem that combines geometry, statistics, and some beautiful mathematical patterns. The question arose: How does the regression line behave when we rotate a regular polygon while keeping its centroid fixed?

The Problem

Consider a regular polygon with n sides. Let's say we have points placed at each vertex of this polygon. If we treat these points as our data points and try to fit a regression line, what happens to this line as we rotate the polygon?

The Mathematical Approach

For a regular polygon with n sides, the vertices can be represented in the complex plane as:

\[ z_k = e^{2\pi i k/n} \text{ for } k = 0, 1, ..., n-1 \]

When we rotate the polygon by an angle Î¸, the new vertices become:

\[ z_k(\theta) = e^{i\theta} \cdot e^{2\pi i k/n} = e^{i(\theta + 2\pi k/n)} \]

The Regression Line

The regression line for these points can be expressed as:

\[ y = mx + b \]

where:

\[ m = \frac{\sum(x-\bar{x})(y-\bar{y})}{\sum(x-\bar{x})^2} \]
\[ b = \bar{y} - m\bar{x} \]

The Beautiful Pattern

What makes this problem fascinating is that for regular polygons, the regression line rotates at exactly half the rate of the polygon's rotation. This can be proven using the properties of complex numbers and the symmetry of regular polygons.

Implications and Applications

This property has interesting implications in:

  • Pattern recognition in geometric data
  • Understanding rotational symmetries in statistical analysis
  • Applications in computer graphics and image processing

Conclusion

This exploration reveals a beautiful connection between geometry and statistics. The fact that the regression line rotates at half the rate of the polygon's rotation is not just a mathematical curiosity but also demonstrates the elegant patterns that emerge when we combine different branches of mathematics.
