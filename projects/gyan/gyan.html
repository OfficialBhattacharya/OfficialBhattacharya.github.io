GyAN

A PyQt5-based desktop application that automatically scrapes, filters, and delivers UPSC-relevant news to your inbox daily.

Project Overview

Desktop Application

Clean, user-friendly desktop interface with real-time status monitoring and live activity tracking.

Multi-Source Scraping

Fetches news from The Hindu, PIB, and Indian Express with robust error handling and retry mechanisms.

Smart Filtering

Uses UPSC-relevant keywords to intelligently filter important news from various sources.

Email Delivery

Sends beautifully formatted daily digest emails with configurable scheduling and multiple SMTP support.

Automated Scheduling

Configurable daily delivery times with background processing and automatic retry mechanisms.

Activity Monitoring

Real-time logs, status updates, and comprehensive monitoring of all application activities.
